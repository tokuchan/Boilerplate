Boilerplate

A Python framework for short, readable, command line programs in Python.

Download .zip Download .tar.gz View on GitHub

Quick Start

from Boilerplate import boilerplate as bp

def my_func(plate, args):
   plate.log.info('Loggers are set up automatically in BP.')
   plate.log.info('Try using --verbose=INFO to see these messages.')
   plate.log.info('plate.args.foo ({}) == args.foo ({}), for your convenience.')
   plate.log.info('Try setting X_FOO, where X is the name of this program\'s file.')

if __name__ == '__main__':
  my_bp = bp.Boilerplate(my_func, 'Test program for Boilerplate.')
  my_bp.parser.add_argument('--foo', 
    action=bp.EnvironmentDefault, 
    environment_variable=self.envbase+'_FOO', 
    help='Sample option.')
  my_bp.main()

Contributing

As you can, no doubt, tell, this project is an infant. There is much to do before I can call it a beta release. Please feel free to fork a copy and play around. If you create something cool, I'll happily take pull requests!

Bug Reports

GitHub offers an issue tracker, so if there is a problem that is not resolved by documentation, you can submit a bug report. I cannot promise to get to them right away, but I will look there first when deciding what to do next with this project.
